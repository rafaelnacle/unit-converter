# Unit Converter

A basic, but cool project for making our life easier by converting common units.

## Features:
It's pretty self-explanatory, but you can convert the following:
- Meter to Feet
- Liters to Gallons
- Kilograms to Pounds

It also shows the other way around! Have fun!
